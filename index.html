Jeu du pendu



- Sélectionner une lettre que vous penssez présente dans le mot. Si celle-ci est presente elle apparaitra dans le mot. Sinon vous perdrer un coups.

- Faite une proposition de mot en cliquant sur le boutton Proposer un mot. Si c'est le bon vous gagnez sinon perdez un coups.

- Vous gagner si votre proposition de mots est la bonne ou si vous cliquer sur toutes les lettres contenu dans le mot. Vous perdez si vous utilisez vos 11 proposition sans trouver le mot.
